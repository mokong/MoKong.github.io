List类型

List是集合类型，声明有几种方式，使用方式可以参考Swift中的Array

  • var myList = List(): 非固定长度的数组
  • var myList = List(2): 长度为2的数组
  • var myList = List<String>(): 创建一个String类型的数组
  • var myList = [1, 2, 3]: 创建一个1、2、3的数组
