Safari中使用NewBing

发表于2023-03-23|更新于2024-01-22
|浏览量:

NewBing 的权限已经有了，但是想在 Safari 中使用 NewBing，因为不想日常开三个浏览器，Safari、Chrome、Edge，电脑内存不允许。。。

首先说步骤：

  1. Safari 安装 Microsoft Bing for Safari插件，并登录，能使用NewBing的账号
  2. 修改 Safari 的 UserAgent 为 Edge 的 UserAgent
  3. 修改默认搜索引擎为Bing

第一步很简单，就打开商店搜索安装即可，安装完成后，打开 bing并登录有NewBing权限的账号；

麻烦的是第二步，网上搜索到大部分是，打开Safari -> Develop -> User Agent -> Microsoft Edge，如下图：
