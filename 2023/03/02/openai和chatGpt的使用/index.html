玩转OpenAI和ChatGpt

发表于2023-03-02|更新于2024-01-22
|浏览量:

背景

之前在其他平台就看到有大佬分享自己的ChatGPT的使用, 比如: 自力hzlzh, 但一直也没弄. 昨天突然看到 OpenAI 在商店有很多应用, 下载了一个发现里面接口都是失败的, 但是界面和方向感觉很有意思, 打算自己做一个自用的.

注册 OpenAI

注册过程参考超详细注册OpenAI接口账号的教程, 很详细, 按照步骤一步步来即可, 要注意的国内的手机号收不到验证码, 所以需要国外的手机号或者通过接码平台(需付费), 验证码的接收我选用的US的手机号, 花费了 7.1元, 一次就成功了.

简单的说:

  • 首先要能打开openAI的注册页面, 然后注册, 到接收验证码界面
  • 如果有国外的手机号, 直接输入, 接收验证码; 如果没有, 再去注册一个接码平台, 选择语言为中文, 然后就都能看懂了, 然后充值 1 美元, 可以用支付宝支付, 然后搜索openai的接码, 选择一个, 耐心等待, 收到分配的号码之后
  • 再去openAI的验证码界面, 输入分配的号码, 发送验证码, 然后等待, 在接码平台就会显示收到的验证码, 输入就可以注册成功.

如何使用OpenAI

注册完成后, 登录进入, 界面如下

对外的使用

点击View API Keys, 可以生成API Key, 可用于接口调用. 接口文档参考Documentation. 需要注意的是, API Key生成后, 要自己保存起来, 只能看到一次.

商店里的那些应用(如下面这些), 可能就是做了一个界面, 然后通过API Key调用Open API 的接口. 相当于是OpenAI的对外使用, 即通过OpenAI对外部用户提供功能.

对内的使用

探讨对内使用之前, 先来看下ChatGPT和OpenAI的关系.

简单的理解, ChatGPT是通过OpenAI训练出来的一个具体应用, OpenAI包含ChatGPT, 而ChatGPT的成功使得OpenAI更加火爆了.

ChatGPT的可以通过Playgroud底部的try it now打开, 如下; 也可以直接打开https://chat.openai.com/.

然后接着来看对内的使用, 即OpenAI如何能帮助自己, 提升自己?

来看下自力hzlzh大佬分享的用法:

给助理分类的小技巧：

  • 点击左侧 Chat 可以重命名
  • 使用 Emoji 方便区分
  • 给 AI “洗脑”的元话术：“在今后的对话中，请你将始终扮演我的 XX 助理/老师，无论我发给你什么内容，你都要以 XX 格式进行输出，无需我再强调。”

把它叫做：垂类小助理，每个助理一个对话框，反复训练它们更懂自己，减少废话和定语，限定回复格式❤️

  • 创意助理：从优劣分析我的Idea
  • 搜索助理：检索信息来源注明链接
  • 代码助理：给出实现思路和代码示例
  • 翻译助理：多国翻译同时直出
  • 文案助理：错别字和润色文案

需要注意两点:

  1. 使用话术, 明确告诉AI, 在当前分类下, 它需要是一个什么角色
  2. 如果窗口关闭或者时间过长, 它可能忘记上下文, 需要再次强调明确告知它在当前分类下要做什么

对于笔者来说:

不知道如何用英文表达, 所以让它作为一个英文老师; 不知道如何写 APP 简介和更新内容, 所以让它提供文案.

所以OpenAI究竟能做什么? 不取决于它, 取决于使用者, 还记得小时候遇到不会的问题时候, 恨不得有个什么都懂的老师时时刻刻在自己身边, OpenAI就是这个什么都懂的老师. 头脑风暴、学习语言、模拟面试、写故事、推荐书籍、规划行程，一切的一切只要能想到的，它都能帮你。

还差一点

一直以来，笔者都有个问题：有一个很棒的主意，知道是什么方面的，功能有什么，但是，但是，不会设计，从而无法把它实践出来。

恨不得有个Police Sketch arist在身边, 我说描述, 它就能画出来.

昨天偶然发现大佬倪爽, 分享了一个全世界第一个使用自然语言/文字进行 UI 设计的设计工具 Galileo, 如下图, 惊为天人, 这就是我心心念念的功能. 根据文字生成设计稿, 绝了, 这样的功能谁不喜欢!

总结

其实在OpenAI刚开始的时候, 我就已经知道了, 毕竟关注了那么多大佬, 之所以迟迟没有行动. 一方面是因为注册验证码麻烦, 另一方面是, 我不知道它能做什么, 以及它能如何帮到我. 所以当我看到商店里那些应用的时候, 我意识到我错了, 能做什么很好的体现出来了, 而能帮我什么, 自力hzlzh已经给了很好的示范. 在我自己体验了之后, 更加觉得相见恨晚. 真心建议大家都能自己注册体验.
